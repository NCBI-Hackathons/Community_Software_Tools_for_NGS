ClinVar Tools on github

Metatoolstest

Abramconnelly

  • Files parsed/used: xml and text
  • Output: Text custom for their workflow and TSV
  • Output columns: Chromosome, Position, Name, Significance, Frequency, Zygosity, ACC URL

Agermanidis

  • A tool for transforming a XML datasource into JSON documents according to a specified schema. The tool lets you define a simple schema in a python dict (schema.py). --> The script uses a simple DSL to specify the fields to extract from the input XML to produce the output.

Alexhnrie

  • ClinVar for Nerds is a web-based interface to query ClinVar data and download it in CSV, JSON, VCF, JSON-LD (RDF model based JSON) format. The project setup downloads the clinvar full xml, parses it into json objects and dumps them in a mongodb instance. After that you can query the data in mongodb. The web interface can take a file format, multiple "restriction" filters as inputs.

ClinGen

  • Demonstration of the genboree.com REST API on a website using a React component to retrieve and display data.

Clingendb

  • The ClinVar database, maintained by the National Center for Biotechnology information (NCBI), provides information about medically important variants with associated phenotypes and effects of sequence changes.

Dakl

  • The MacArthur lab has made a great effort make ClinVar data more available to the bioinformatics community. This script converts their flat file to a VCF file.

Ianjw11

  • to run converter script: python run.py --source /path/to/clinvar.xml --output /path/to/output.json

MacArthur

  • This repo provides tools to convert ClinVar data into a tab-delimited flat file, and also provides that resulting tab-delimited flat file.

PersonalGenomesOrg

  • Generate a report based on matching a genome VCF to the VCF version of ClinVar's variant database. (possibly related to https://github.com/abramconnelly/clinvar-report, same team etc.) This code is intended to be used as a module (installable via pip), for use in other systems or pipelines.

Pidr

Rbagnall

  • Used to format Clinvar pathogenicity predictions into a snp hash table and indel hash table. Puts mutliple predictions for the same nucleotide into one line. Usage clinvar_parser.pl variant_summary.txt.
